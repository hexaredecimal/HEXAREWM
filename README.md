
# HEXAREWM
>> MY Personal Lightweight tiling window manager for linux using penrose
## Requirements
- Linux/Unix based
- Rust toolchain

## Build
```sh
$ git clone git@github.com:hexaredecimal/HEXAREWM.git
$ cd HEXAREWM
$ cargo build --release
```
